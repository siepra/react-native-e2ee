# react-native-e2ee

End-to-End Encryption (E2EE) made easy for react-native projects

## Installation

```sh
npm install react-native-e2ee
```

## Usage

```js
import { multiply } from 'react-native-e2ee';

// ...

const result = await multiply(3, 7);
```
